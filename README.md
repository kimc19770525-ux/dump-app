# 덤프지입 일보 관리 앱

## Vercel 배포 방법 (5분)

### 1단계 — GitHub 업로드
1. [github.com](https://github.com) 가입 (없으면)
2. 오른쪽 상단 **+** → **New repository**
3. Repository name: `dump-app` 입력 → **Create repository**
4. **uploading an existing file** 클릭
5. 이 폴더 안의 **모든 파일과 폴더**를 드래그해서 업로드
6. **Commit changes** 클릭

### 2단계 — Vercel 배포
1. [vercel.com](https://vercel.com) 접속 → **Sign up** → GitHub으로 로그인
2. **Add New Project** 클릭
3. 방금 만든 `dump-app` 선택 → **Import**
4. 설정 그대로 → **Deploy** 클릭
5. 1~2분 후 링크 생성 완료!

### 완료!
- 링크 예시: `https://dump-app-xxx.vercel.app`
- 이 링크를 카톡 단톡방에 공유하면 기사들이 바로 접속 가능
- 기사들은 가입 없이 링크만 누르면 일보 입력 가능

## 주의사항
- 데이터는 각 기기의 브라우저(localStorage)에 저장됩니다
- 관리자(철수씨)는 항상 같은 기기/브라우저에서 접속해야 데이터가 유지됩니다
